Certificate

To lay the foundation for HCI research, I worked on a project to build applications where computers and humans interact. I learned how to develop AI models for three types of interfaces--visual, language, and voice--through NVIDIA DLI, and I earned a certificate upon completing the course.

Building Conversational AI Applications

The development of the AI model for the voice interface began with the preprocessing of human voice data. After preprocessing and feature extraction from the voice data, I built a speech recognition (SR) model. The system generates responses to the input speech using a language model, which are then delivered back to the user through text-to-speech (TTS) technology.

Building Transformer-Based NLP Applications

I built a language model based on a Transformer architecture, training it on corpus data to generate appropriate responses to the input language.

Computer Vision for Industrial Inspection

With the goal of developing computer vision models for industrial applications, I built a model that analyzes images of PCB boards to identify faulty soldering.
